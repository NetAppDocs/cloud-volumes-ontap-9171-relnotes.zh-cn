= 页面标题显示在此处
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
将此模板用于文章、主题，无论您如何称呼它们。index.adoc 文件不具有文章页面的正确关键字，将其用作模板可能会在以后导致发布问题。


NOTE: 在初始发布之前删除此文件，因为即使此文件未链接到侧边栏，也将发布此文件。创建了一些文章后，您可以将其用作入门文章。

如果您的文章将位于内容子文件夹中，则需要对上面显示的符号进行两次调整：

* `:imagesdir:../media/`
* `permalink: folder-name/sample-article.html`
